# Veri Analizi Projesi (VA)

Bu proje, kapsamlı bir veri analizi çalışmasını içeren Jupyter Notebook ve rapor dosyalarını barındırır.

## İçerik

*   **Jupyter Notebook (.ipynb):** Veri temizleme, işleme, analiz ve görselleştirme adımlarını içeren Python kodları.
*   **Proje Raporu (.docx):** Analiz sonuçlarının, metodolojinin ve bulguların detaylı açıklaması.

## Kullanılan Teknolojiler

*   Python
*   Pandas, NumPy (Veri İşleme)
*   Matplotlib, Seaborn (Veri Görselleştirme)
*   Jupyter Lab / Notebook

## Nasıl Çalıştırılır?

1.  Bu repoyu klonlayın.
2.  Gerekli kütüphanelerin yüklü olduğundan emin olun (`pip install pandas matplotlib ...`).
3.  `.ipynb` dosyasını Jupyter Notebook veya VS Code üzerinde açarak hücreleri çalıştırın.
